# 🔐 Credentials Generator

> ✨ A lightweight, zero-dependency Python script that instantly generates cool, randomized tech-themed usernames and strong, secure passwords completely offline! 🤖💻

---

## 🚀 Features

* 📊 **Multi-Option Output:** Generates 3 unique account combinations at once so you can choose your favorite!
* 🔒 **Strong Passwords:** Creates robust 14-character passwords using uppercase/lowercase letters, numbers, and secure special characters (`!?@#$`).
* 🔌 **100% Offline & Safe:** Zero internet connection or external APIs required! Completely bypasses all firewall or antivirus blocks (like Kaspersky).
* 📦 **Zero Dependencies:** Uses only Python's built-in libraries (`random` and `string`). No need to run `pip install`!

---

## 🛠️ Requirements & Installation

* 🐍 **Python 3.10+** (Highly recommended: Stable **Python 3.12** 🚀)
* ❌ No external libraries needed! Just pure Python power! 🔥

---

## 💻 How to Use

1. 📂 Clone or download this repository to your PC.
2. 🖥️ Open your terminal or Command Prompt (CMD) and navigate to the project folder:
   ```bash
   cd path/to/folder
   ```
3. 🏃‍♂️ Run the script using the following command:
   ```bash
   python gen.py
   ```
